# LemonClaw 产品方案

## 一、产品简介

**LemonClaw** 是一个面向非技术用户的 AI Agent 桌面应用。它将底层强大的 AI Agent 运行时 **OpenClaw** 封装成了人人都能使用的图形化驾驶舱，内置 **HomiClaw** 同款的企业级 Fallback 与 Session 压缩机制以保证稳定可靠，并首次引入了源自 **Hermes** 的 **Agent 自我进化闭环（Agent Self-Evolution Loop）**——让 AI 助理能在服务你的过程中自动复盘、沉淀经验、升级技能，实现越用越聪明。

> **一句话定位**：LemonClaw 将底层 AI Agent 运行时 OpenClaw 封装为人人可用的桌面驾驶舱，内置 HomiClaw 同款的企业级稳定机制，并注入 Hermes 的 Agent 自我进化闭环——零代码配置，越用越聪明，越用越稳。

## 二、核心特性

LemonClaw 集三家之长，在易用性、稳定性和自我成长能力上做到了独特的平衡。

### 2.1 源自 RivonClaw：零代码桌面驾驶舱

| 特性 | 说明 |
| :--- | :--- |
| **图形化桌面应用** | 基于 Electron 构建，系统托盘常驻，所有配置在可视化面板中完成，无需打开终端。 |
| **自然语言规则** | 用自然语言描述“当收到什么消息时该做什么”，系统自动编译为可执行策略，热重载、零重启。 |
| **多模型多渠道** | 原生支持 20+ 主流大模型服务商，打通 Telegram、微信、飞书等 10+ 消息渠道。 |
| **本地优先，隐私自有** | 所有数据存储在本地，API 密钥系统级加密，隐私不泄露。 |

### 2.2 源自 HomiClaw：企业级稳定可靠

| 特性 | 说明 |
| :--- | :--- |
| **LLM Fallback 机制** | 主模型故障时自动切换备用模型，冷却期保护防止雪崩，保证 Agent 始终在线。 |
| **Session 自动压缩** | 对话过长 Token 超限时自动压缩历史，保留关键决策与待办事项，确保长会话不中断。 |
| **工具审批与安全沙箱** | 危险操作需用户确认，敏感路径写保护，兼顾能力与安全。 |
| **打包式 Gateway** | 运行时直接嵌入应用，比独立进程响应更快。 |

### 2.3 源自 Hermes：Agent 自我进化闭环

| 特性 | 说明 |
| :--- | :--- |
| **Agent Self-Evolution Loop** | 每完成一次任务自动触发复盘流程：分析结果 → 提炼经验 → 沉淀技能，形成「执行→复盘→沉淀→调用」的完整闭环。 |
| **持久化记忆与技能库** | 用户偏好、常用工作流向量化存储于本地技能库，下次遇到相似任务自动调用最优技能。 |
| **技能自动升级** | 同一技能在多次执行中根据成功率与用户反馈自动优化执行策略，无需人工干预。 |
| **可视化进化轨迹** | 管理面板提供技能沉淀时间线、成功率曲线和 Token 成本变化图，直观看到 AI 助理的成长。 |

## 三、与参考项目的定位对比

LemonClaw 并非从零开始，而是站在巨人肩膀上，精心组合了三款优秀产品的核心能力。

### 3.1 维度对比表

| 维度 | RivonClaw | HomiClaw | Hermes | **LemonClaw** |
| :--- | :---: | :---: | :---: | :---: |
| **易用性封装** | ✅ 强 | ✅ 强 | ❌ 无 | ✅ 继承 RivonClaw |
| **企业级稳定性** | ❌ 无 | ✅ 强 | ❌ 无 | ✅ 继承 HomiClaw |
| **自进化能力** | ❌ 无 | ❌ 无 | ✅ 强 | ✅ 继承 Hermes |
| **目标用户** | 个人 | 企业内部 | 开发者 | **追求稳定+成长的个人** |

## 四、产品差异化价值总结

| 用户痛点 | RivonClaw | HomiClaw | Hermes | **LemonClaw** |
| :--- | :---: | :---: | :---: | :---: |
| 不会写代码，想用 AI Agent | ✅ 解决 | ✅ 解决 | ❌ 需要编程 | ✅ 解决 |
| 担心 AI 不稳定，关键任务中断 | ❌ 无保障 | ✅ 有保障 | ❌ 无保障 | ✅ 有保障 |
| AI 总记不住我的习惯，每次都要重新教 | ❌ 无记忆 | ❌ 无记忆 | ✅ 有记忆 | ✅ 有记忆 |
| 希望 AI 越用越聪明，自己成长 | ❌ 不支持 | ❌ 不支持 | ✅ 支持 | ✅ 支持 |

**LemonClaw 是首个同时解决「易用性」「稳定性」「持续成长」三个核心诉求的桌面 AI Agent 产品。**
