## 📝 编码规范

### TypeScript
- 使用接口定义类型
- 函数添加返回类型
- 避免 `any`，用明确类型

### React
- 函数组件 + Hooks
- 状态用 useState
- 副作用用 useEffect

### 样式
- 使用 CSS 类名
- 不要行内样式（除动态颜色）
- 保持响应式
