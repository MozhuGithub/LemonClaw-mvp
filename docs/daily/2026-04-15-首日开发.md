## 🐛 解决的问题

### 问题 1：Electron 窗口不显示
**症状**：进程启动成功但看不到窗口  
**原因**：窗口被其他程序遮挡或在后台  
**解决**：`pkill -f lemonclaw-mvp` 清理后重启

---

### 问题 2：TypeScript 编译错误
**症状**：`Cannot find module './core/agent/AgentManager'`  
**原因**：导入路径错误（`./` vs `../`）  
**解决**：修正为 `../core/agent/AgentManager`

---

### 问题 3：package.json 没有 build 命令
**症状**：`pnpm build` 报错 `Command "build" not found`  
**原因**：scripts 只有 `build:ts` 和 `build:vite`  
**解决**：添加 `build` 命令运行 `npm-run-all build:ts build:vite`

---

### 问题 4：ipcMain 未定义
**症状**：`ReferenceError: ipcMain is not defined`  
**原因**：修改 main.ts 时漏掉导入语句  
**解决**：重新写入完整的 main.ts，包含正确的导入

---

### 问题 5：代码格式显示问题
**症状**：AI 回复的代码没有格式  
**解决**：
- 添加 `white-space: pre-wrap` 保留换行
- 添加等宽字体和代码块背景
- MVP 阶段足够使用（暂不需要语法高亮）

## 📊 技术决策

### 决策 1：MVP 不用 Markdown 渲染
- **原因**：语法高亮增加复杂度
- **方案**：用 CSS `white-space: pre-wrap` 保留格式
- **效果**：代码可读性够用，后续可升级

### 决策 2：热加载用 nodemon
- **原因**：开发效率优先
- **方案**：`tsc --watch` + `vite` + `nodemon` 并行
- **效果**：改代码自动刷新，无需手动重启

### 决策 3：Agent 配置内存存储
- **原因**：MVP 阶段简化
- **方案**：暂用内存存储，后续加 SQLite

## 💡 学到的经验

1. **Electron 路径问题**：主进程导入用相对路径，注意 `../` vs `./`
2. **热加载配置**：`nodemon` 监听 `dist/main/` 而非源码
3. **CSS 文本溢出**：必须加 `min-width: 0` 才能让 `text-overflow` 生效
4. **pnpm 依赖**：新依赖要 `pnpm install` 后才能用
5. **版本管理**：v0.1.0 → v0.2.0 标记多 Agent 升级
